# Life Dashboard

## 概要

Life Dashboard は、日々の行動や体調を最小限の操作で記録し、継続的な振り返りを支援する個人向けアプリケーションです。

本プロジェクトは単なる個人開発ではなく、要件定義から運用設計までを一貫して実践することを目的としています。

---

## 開発目的

以下のスキルを実践的に身につけることを目的とします。

* 要件定義
* 基本設計
* 詳細設計
* フロントエンド開発
* バックエンド開発
* AWSを用いたインフラ構築
* Infrastructure as Code
* テスト設計・実装
* CI/CD構築
* 運用設計

※競合分析などの既存プロダクト調査は実施しない

---

## 解決したい課題

日々の行動や体調を記録したいと思っても、

* 入力が面倒
* 継続できない
* 振り返らない

という問題があります。

本アプリでは「3秒以内で記録できること」を重視し、継続しやすい仕組みを目指します。

---

## MVP（初期リリース）

### 記録機能

* 勉強時間
* 筋トレ
* 読書
* 睡眠
* 気分

を簡単に記録できる。

### 閲覧機能

* 今日の記録
* 週間集計
* 月間集計

を確認できる。
